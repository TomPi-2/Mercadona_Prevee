# Mercadona_Prevee  
**App predictiva para anticipar las compras futuras del usuario.**

Mercadona_Prevee es una aplicación desarrollada en **Flutter** que combina **inteligencia artificial, datos y diseño moderno** para ayudar al usuario a **predecir, planificar e informarse** sobre sus próximas compras.  
Su objetivo es ofrecer una experiencia personalizada basada en **gustos, stock disponible, ubicación y ofertas** actuales.

---

## Características principales
- **Predicción inteligente** de productos basados en historial y preferencias.  
- **Consulta de stock en tiempo real** por tienda o zona.  
- **Recomendaciones personalizadas** según los hábitos del usuario.  
- **Ofertas dinámicas** que se adaptan a los gustos detectados.  
- **Interfaz moderna y responsiva** desarrollada en Flutter.

## Intrucciones
1. Abrir Visual Studio Code
2. Clicar en Clone Git Repository
3. Seleccionar Mercadona_Prevee

## Ejecutar
En terminal de Visual Studio Code (powershell):
- Limpiar proyecto (opcional)
```bash
flutter clean
```
- Obtener dependencias (si se ha hecho flutter clean)
```bash
flutter pub get
```
- Ejecutar en el navegador Chrome:
```bash
flutter run
```
- Seleccionar Chrome (opción 2)
